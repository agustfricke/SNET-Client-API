# httpx
⚡ Lightweight client API application developed with Go!

### Features
- :satellite: JWT 
- :satellite: File support
- :satellite: GET 
- :satellite: POST 
- :satellite: PUT 
- :satellite: DELETE

### install and use in linux:

```bash
git clone https://github.com/agustfricke/httpx.git
```

```bash
cd httpx
```

```bash
sudo mv main /usr/local/bin/httpx
```

```bash
source .zshrc
```

```bash
httpx
```
