# snet-client-api
⚡ Lightweight client API application developed with Go!

### Features
- :satellite: JWT 
- :satellite: File support
- :satellite: GET 
- :satellite: POST 
- :satellite: PUT 
- :satellite: DELETE

### install and use in linux:
```shell
git clone https://github.com/agustfricke/snet-client-api.git
```
```shell
cd snet-client-api
```
```shell
sudo mv main /usr/local/bin/sn
```
```shell
source .zshrc
```
```shell
sn
```
